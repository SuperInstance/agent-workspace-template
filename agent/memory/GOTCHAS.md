### Format new entries like:

```markdown
## YYYY-MM-DD: [short title]

**Area:** [module, dependency, or subsystem]
**Symptoms:** [what broke or confused]
**Root cause:** [why it happened]
**Fix:** [what fixed it]
**How to avoid:** [what to watch for in the future]
```

*Experience costs time. Recording it saves time for the next agent.*
