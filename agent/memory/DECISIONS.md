# Design Decisions

> Every significant design decision, recorded with rationale and alternatives considered.

Each entry includes:
- **Decision:** What was chosen
- **Rationale:** Why this path
- **Alternatives:** What was rejected and why
- **Date:** When the decision was made
- **Session:** Which agent session generated it

### Format

```markdown
## [Decision title]

- **Date:** YYYY-MM-DD
- **Session:** [session reference]

**Context:** [what problem was being solved]

**Decision:** [what was chosen]

**Rationale:** [why this was the right choice]

**Alternatives considered:**
- [Alternative A]: [why rejected]
- [Alternative B]: [why rejected]

**Trade-offs accepted:** [what was compromised]
```

---

*Decisions live here, not in Slack, not in email, not in someone's head.*
